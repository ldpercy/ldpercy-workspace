### Resist Fingerprinting

	privacy.resistFingerprinting = true

Resist fingerprinting spoofs timezone, so will break some pages
https://support.mozilla.org/en-US/questions/1297208



### JavaScript in PDFs

To turn off JavaScript execution in PDFs: Enter about:config in the address bar; click "I'll be careful." In the search box near the top, enter pdfjs.enableScripting. Change the setting to False.
