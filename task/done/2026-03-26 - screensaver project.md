Screensaver project
===================


Get a screensaver project going to show off some css animations.



I've been enjoying adding animations to the polygon app, but it's drifting a bit out of its lane so have been wondering if it should be broken out into its own project.

It also occurs to me that while the animations look quite neat on polygons & polygrams, the techniques would apply to pretty much any shape I can conjure up.

So thinking instead about getting a screensaver project going where I can be a bit freer and do things just for fun, be creative etc.

Will also be helpful in figuring out how to better apply and combine styles & animations dynamically.
