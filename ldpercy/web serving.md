Web serving
===========


Python http server
------------------

https://docs.python.org/3.9/library/http.server.html


Currently using this for local needs:

```bash
#!/bin/bash
python3 -m http.server 9001 --bind 127.0.0.1
```


> By default, the server uses the current directory. The option -d/--directory specifies a directory to which it should serve the files.
